What is Redis Cluster ?

大家應該會了解Redis原先的一些特性，Cluster是他的重要功能之一，Redis中設計了HashSold，總共有16384個HashSolt，當一個Cluster建立時，將會把這些Solt分配到這些Node中，依照官方說明，若有3個node，各節點分配依序為：
Node 1: 0 ~ 5500
Node 2: 5501 ~ 11000
Node 3: 11001 ~ 16384
後續的事情，基本上Redis會自行決策要使用哪個Solt!!
