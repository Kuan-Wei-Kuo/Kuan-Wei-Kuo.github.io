What is Spring Security ?

Spring Security以J2EE為基底，為企業軟體提供安全服務。Spring Security專注於提供身分驗證與認真，因此提供了很多種方式的認證機制。

Get Started

對於Security我沒有想法，我只能說…依照場景去分析自己該使用的機制，這邊簡單做個登入機制給各位看看。

Java

SecurityConfig.java

這邊就是主要的設定檔，將在這邊認證相關設定。

SecurityInitializer.java

這邊要記得將這個類別啟動，否則會沒辦法運行Security功能。

LoginController.java

JSP

home.jsp

變複雜許多，因為我加入了SecurityTag，尚未登入時會顯示Login按鈕，登入後將顯示使用者名稱以及登出按鈕。

login.jsp

基本上FormLogin欄位大同小異，需要注意的為登入後導向以及form action位置。

403.jsp
